# Fauxnix Threads

Threads are named workspaces plus app launch profiles. On the
Fauxnix-Archivist branch, the active profile is Wayfire with a Chromium kiosk
browser pointed at the local node desktop. Ordinary desktop apps, Faux-pass
providers, and the workspace canvas can still be launched from the local kiosk
or terminal.

Useful commands:

- `faux-pass status`
- `faux-pass apps`
- `faux-pass run web`
- `faux-pass run terminal`
- `faux-pass run fennix`

Legacy thread definitions may remain here for reference, but the ThinkPad
file-server build should prefer the current Wayfire workspace and Archivist
tooling path.
